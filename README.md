# Unsplash Explorer

## Features:

- Search for users
- View user details
- View user's photos

## Tech stack:

- React
- Styled Components
- React Router

## Installation and build instructions:

To run dev server, run `yarn install && yarn start`

To build and get files to deploy, run `yarn install && yarn build`
